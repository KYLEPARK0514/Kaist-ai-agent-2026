# KAIST AI Agent - Client Web App

This directory contains the Phase 3 web application for the KAIST AI Agent, built with React, Vite, TypeScript, and Tailwind CSS.
It is intended to be deployed to Azure Static Web Apps.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Key Technology Stack

- **React 18** for UI
- **Vite** for build tooling
- **TypeScript** for static type checking
- **Tailwind CSS** for styling
